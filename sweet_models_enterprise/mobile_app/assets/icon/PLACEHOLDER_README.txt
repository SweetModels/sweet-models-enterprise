# Ícono Placeholder - Sweet Models

Este es un archivo de texto placeholder para `app_icon.png`.

Para generar un ícono real, puedes usar estas opciones:

## Opción 1: Crear con Canva (Rápido)

1. Ve a https://canva.com
2. Crea diseño personalizado: 1024x1024
3. Diseña tu logo con estos colores:
   - Background: #09090B (negro)
   - Accent: #00F5FF (cyan)
   - Secondary: #EB1555 (rosa)
4. Descarga como PNG
5. Guarda aquí como `app_icon.png`

## Opción 2: Usar un generador

1. Ve a https://icon.kitchen
2. Sube una imagen o crea un ícono
3. Personaliza los colores
4. Descarga todos los tamaños
5. Guarda aquí

## Herramienta online rápida

https://www.photopea.com/ (gratis, como Photoshop online)

1. Nuevo archivo: 1024x1024
2. Rellena con negro (#09090B)
3. Agrega texto "SM" centrado
4. Color del texto: cyan (#00F5FF)
5. Exporta como PNG
6. Guarda aquí como `app_icon.png`
