## üîå Backend Requirements

Your backend **MUST** have 3 endpoints:

### 1. **GET /admin/dashboard**
Returns dashboard statistics
```json
{
  "total_tokens": 50000,
  "total_revenue_cop": 2500000.50,
  "active_models": 12,
  "alerts_count": 0,
  "last_updated": "2024-01-15T14:30:00Z"
}
```

### 2. **GET /admin/live-activity**
Returns list of active streams
```json
[
  {
    "model_name": "Alejandra Vega",
    "room_name": "Private Show",
    "viewers_count": 42,
    "status": "active",
    "started_at": "2024-01-15T14:15:00Z"
  }
]
```

### 3. **GET /admin/alerts**
Returns system alerts
```json
[
  {
    "type": "warning",
    "title": "High Payout",
    "message": "Model xyz has high payout request",
    "is_read": false,
    "created_at": "2024-01-15T14:25:00Z"
  }
]
```

All endpoints require: `Authorization: Bearer {jwt_token}`
